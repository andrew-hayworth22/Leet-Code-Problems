# Median of Two Sorted Arrays
## Problem
Given two sorted arrays, return the median of the two sorted arrays
## Example
Input: nums1 = \[1,3\], nums2 = \[2\]
Output: 2.00000
## Constraints
* nums1.length == m
* nums2.length == n
* 0 <= m <= 1000
* 0 <= n <= 1000
* 0 <= s.length <= 5 * 10<sup>4</sup>
* -10<sup>6</sup> <= nums1\[i\], nums2\[i\] <= 10<sup>6</sup>
